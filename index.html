Braille is a notational system developed in the early 1800s by Frenchman Louis Braille. Notational systems represent the sounds or words of a language. In the case of Braille, the notational system is tactile instead of visual.

Each letter of the alphabet is encoded in a single braille cell. In English Braille, this cell is arranged using six dots in a 2 by 3 matrix.

In Braille, each letter is derived from the Latin alphabet, resulting in a dot pattern system that corresponds to alphabetic letting order within the French alphabet.
